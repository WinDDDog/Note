所谓栈帧格式，就是java运行方法时的局部变量储存。。。。算了

每个线程有自己独立的栈帧

```
/////////////////
/////局部变量/////
/////////////////
/////操作数栈/////
/////////////////
/////动态链接/////
/////////////////
/////返回地址/////
/////////////////
——————————————————
/////上一个栈/////
/////局部变量/////
```

## 操作数栈

用于做计算的栈

比如: a+b  ->  addl a,b ->( push a; push b; addl; )

也会被用来进行参数传递

因此有些引擎被称为“基于栈的执行”

## 动态链接

指向常量池的方法引用
